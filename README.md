# 🏥 Diabetes Prediction System

An AI-powered web application that predicts the likelihood of diabetes using machine learning.

Built using:
- Python
- Scikit-learn
- Streamlit
- NumPy
- Joblib

---

## 📌 Project Overview

GlucaCheck AI is a machine learning-based web application designed to assess diabetes risk using clinical health parameters.

The model analyzes patient input data such as glucose level, BMI, age, insulin levels, and more to predict whether a person is at high or low risk of diabetes.

This project demonstrates:
- End-to-end ML pipeline
- Model training and preprocessing
- Feature scaling
- Web deployment using Streamlit
- Clean UI with custom styling

---

## 🧠 Machine Learning Model

- Algorithm Used: Logistic Regression / Random Forest (update based on your model)
- Dataset: Pima Indians Diabetes Dataset
- Preprocessing:
  - Handling missing values
  - Feature scaling using StandardScaler
- Model Evaluation Metrics:
  - Accuracy
  - Precision
  - Recall
  - Confusion Matrix

---

## 📊 Input Features

The model uses the following features:

1. Pregnancies  
2. Glucose  
3. Blood Pressure  
4. Skin Thickness  
5. Insulin  
6. BMI  
7. Diabetes Pedigree Function  
8. Age
